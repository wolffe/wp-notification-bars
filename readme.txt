== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `wp-notification-bars` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create Notification Bar: Plugin will add a new menu item called 'Notification Bar'
4. Create New Notification Bar from there.
5. You're done!
